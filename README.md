# Chao-Pi-Adventure
A Tamagotchi like based on the old "Chao Adventure" VMU for Dreamcast.
It uses a Rapsberry Pi Zero and an Adafruit 1.3' OLED Bonnet.

Need to install [Luma core](https://github.com/rm-hull/luma.examples)
